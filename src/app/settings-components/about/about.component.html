Bolero is a self-hosted, visually stunning start page designed to bring elegance, order, and power to your digital workspace. Inspired by the crescendo of Ravel’s famous composition, Bolero transforms the browser homepage into a crescendo of form and functionality—intuitive, organized, and deeply customizable.

Built using Angular and NestJS, Bolero features a robust full-stack architecture backed by SQLite for efficient local data storage. Every element—from categories and link groups to icons, themes, and background images—is designed with user experience in mind, offering a seamless interface for managing, styling, and launching your most-used web resources.

Bolero structures your links in a three-tier hierarchy:

  • Link Categories: Broad organizational groups (e.g., Work, Personal, Dev Tools).
  • Link Groups: Subsections within categories, useful for grouping by project, topic, or tool type.
  • Links: Each link includes a title, URL, description, icon, and tags—fully editable and reorderable via intuitive drag-and-drop.
  • Drag-and-Drop Reordering: Reorder categories, groups, and links with ease.
  • Material Design UI: Smooth animations, clean layout, and responsive controls.
  • Icon Management: Upload your own 120x120 PNG icons, preview them, edit metadata, and apply instantly.
  • Theme Import/Export: Save your visual configuration as a reusable ZIP file. Import a theme to instantly transform your dashboard.
  • Full Backup/Restore: Export all data (SQLite DB + icons + images) into a .zip file, or restore from one. Perfect for backups or migrating to another device.
  • Search with External Providers: Instantly launch a search with a custom provider like DuckDuckGo or Kagi, configurable in your settings.
  • Status Feedback: Clear, visual status messages let you know when actions are processing, succeed, or fail—no guesswork needed.

Bolero was created for developers, tinkerers, and productivity enthusiasts who want their start page to be as powerful as the tools it launches. Every interaction has been crafted to make your workflow faster, cleaner, and more enjoyable.

Bolero is open-source and available on GitHub.
