This will be your main operating area. From this page you can add Categories, Link Groups, and Links. Within link groups, links can drag-and-drop within the same group, or to other groups. To drag a link, click the    icon on the bottom group toolbar, which will enable drag-and-drop. Then use the drag handle to drag the link to where you want it. Using the same feature, you can drag entire groups to different locations under the same category.

To move a group to another category, click the menu at the top right of the group, then go to Move Group, then select the Category you want to move the group to. Other useful features can be accessed from the many menus available.

To access Bolero's settings, click the  button found below the groups area. Other quick start tutorials can be found there.

Using Bolero's Search Bar

You can configure Bolero’s search feature to work with any search engine that supports URL-based searches. Bolero will automatically append your search terms to the end of the URL you provide.

Go to the search engine you want to use, perform a sample search, then in the address bar, copy the full URL up to (but not including) your actual search terms. Then paste that URL into the "URL Query" field, then click save, and Bolero will take care of the rest.

Examples:

  • Google: https://www.google.com/search?q=
  • Duckduckgo: https://duckduckgo.com/?t=h_&q=
  • Yahoo: https://search.yahoo.com/search?p=

A Note on Icons

Optimal icon size is 120x120, and icon format must be .PNG. Using other dimensions can cause render issues, slow loading times, or other unpredictable behavior.

You can use Canva, Photoshop, Adobe Express, or any number of other free or paid image editors to create additional icons for Bolero.

Customizing Your Theme

Use this panel to fully customize the appearance of your dashboard. You can change fonts, colors, background styles, and more—then save your work as a new theme or overwrite an existing one.

A Note on Imports

Importing a backup .ZIP file will erase all current Bolero settings and replace them with what's in the file. This includes links, categories, icons, images, and themes.

Make sure you've exported your current data before proceeding.
