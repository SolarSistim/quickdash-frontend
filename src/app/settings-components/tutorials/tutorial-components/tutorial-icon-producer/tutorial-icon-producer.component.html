Creating Icons

Use the Icon Producer to grab icons from the internet, and then cropping & scaling them here to make them compatible with Remembr icons.

NOTE: Using transparent PNG source images is recommended to maintain visual continuity on the Remembr dashboard. Using non-transparency images like JPG or BMP results in icons without transparent sections.

  • 1. Use Google to find an image you want to convert into a Remembr icon.
  • 2. Save that image to your computer.
  • 3. Upload that image here by selecting "Upload Icon".
  • 4. Crop the image as you see fit.
  • 5. Select the size of the icon. Recommended for Remembr is 128x128.
  • 6. Click "Download PNG".
  • 7. Go to the Icon Manager tab on this page to upload your new icon.
